📘 Exercícios de Lógica e Testes em Java
📖 Descrição

Este repositório reúne exercícios práticos desenvolvidos em Java, com foco no aprimoramento da lógica de programação e na aplicação de testes automatizados. O objetivo é consolidar fundamentos essenciais para o desenvolvimento de software, incluindo resolução de problemas, organização de código e validação de funcionalidades.

🎯 Objetivos
Desenvolver raciocínio lógico e resolução de problemas
Praticar conceitos fundamentais de Java
Aplicar boas práticas de programação
Implementar testes unitários para validação de código
🛠️ Tecnologias Utilizadas
Java
JUnit (para testes)
